 #SO
 Trabalho de Sistemas Operativos 2015/2016.
 Sistema de Backup eficiente.
 
 ##Instalação
 ```sh
 $ make
 $ make install
 ```
 
 ###Desinstalação
 ```sh
  $ make uninstall
 ```
 
 ##Exemplos
 Correr o servidor:

 ```sh
  $ sobusrv 
 ```
  
 Backup do ficheiro a.pdf:
 ```sh
  $ sobucli backup a.pdf
 ```
 
 Restore do ficheiro a.pdf:
 ```sh
  $ sobucli restore a.pdf
 ```
